# 순이와의 생활분석

## Application Deployed

-   https://pl-soony.herokuapp.com/

## How to Start Locally

### Install Dependencies

```bash
$ npm install
```

### Start the app

```bash
$ npm run start
```

### Open the app

##### App running on

-   http://localhost:3000
